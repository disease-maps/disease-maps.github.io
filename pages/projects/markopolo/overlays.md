# Overlays
## Guide for adding data overlays to ExposomeMap-PM

This page offers step-by-step instructions on uploading a custom data overlay to ExposomeMap-PM on the MINERVA platform, covering file preparation and upload process. A general guide to the MINERVA Platform is available at [https://minerva.pages.uni.lu](https://minerva.pages.uni.lu/).

### Step 1. Create an account

To create an account, please contact the MINERVA team at [minerva@uni.lu](mailto:minerva@uni.lu). Optionally, you can use your ORCID ID to create an account. Normally your new account will be activated within 24 hours.

To login, click the icon in the top-left corner.

<img src="/pages/projects/markopolo/images/overlays02.png" width="600" />

Next, sign in via ORCID and submit.

<img src="/pages/projects/markopolo/images/overlays03.png" width="600" />

### Step 2. Prepare data file

The data file should be in txt format and consist of two columns "name" and "value", where "name" column inludes protein/RNA/gene names as in HUGO Gene Nomenclature Committee and "value" column countains values from -1 to 1. 

<img src="/pages/projects/markopolo/images/overlays01.png" width="400" />

### Step 3. Upload data file to overlays

Click the blue "Overlays" button on the top-middle of the screen. Click "Add overlay".

<img src="/pages/projects/markopolo/images/overlays04.png" width="600" />

In the next tab, chose your file and create file name.

<img src="/pages/projects/markopolo/images/overlays05.png" width="600" />

Successfuly uploaded file will be visible as a new overlay. 

<img src="/pages/projects/markopolo/images/overlays06.png" width="600" />

Now it is possible to see the dataset on the map by chosing the corresponding overlay.

Optionally, this overlay can be made public.

<img src="/pages/projects/markopolo/images/overlays07.png" width="600" />

More information to data upload is available in the [MINERVA User's Manual](https://minerva.pages.uni.lu/manuals/v20.0/user/visualisation/#overlays).
